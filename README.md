# school-manager
a dashboard for teachers and managers to keep track of students' attendance and learning performance daily, monthly, quarterly and yearly.
a dashboard for accountant to keep track of financials quarterly and yearly 
# GitFlow
- fork the repo
- git pull origin main
- git checkout -b ‘branch name’ //create a new branch 
- git add -A
- git commit -m "your message"
- git pull origin main
- git merge main //merge everything from main branch 
- git push origin ‘branch name’
- make a pull request

# Frontend

> Architecture
- Nextjs
- Redux Toolkit
- TailwindCss
- DaisyUi
  

> Install dependencies && Start developing
- cd frontend
- npm i
- npm run dev

# Backend
> Architecture
- NodeJs
- ExpressJs
- Sequelize ORM
- Postgresql

> Install dependencies && Start developing
- cd backend
- npm i
- npm run dev
